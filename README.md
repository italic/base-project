# [MJML](https://mjml.io)


## Installer MJML et gulp si ce n’est pas déjà fait

```bash
sudo npm i -g mjml gulp-cli
```

## Installer le projet

```bash
sudo npm i
```

## Travailler !

```bash
npm run start
```

Lance automatiquement un premier build, puis un `gulp watch`
